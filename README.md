# 基于SSM的个人博客管理系统


## 1、项目介绍

基于SSM的个人博客管理系统4实现了博客的发布、评论的审核管理、友链管理等功能


## 2、项目技术

后端框架：SSM（Spring、SpringMVC、Mybatis）

前端技术：Bootstrap、jsp、css、JavaScript、JQuery

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA、Eclipse都可运行
- tomcat版本：Tomcat 7-10版本均可
- 数据库类型：MySql5.x
- maven项目：否
- 硬件环境：Windows 或者 Mac OS
